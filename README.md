# MercadoLibre UX Test
Here you can find the source code of the Mercadolibre UX Test.
In the src folder,you will find the files to develop.
in the dist folder, you will find the minified files version.

[Live Demo]

Develop:
```sh
$ grunt watch
```
Build:
```sh
$ grunt build
```
Technologies Used:
  - NodeJS
  - Grunt
  - Bower
  - HTML5
  - CSS3
  - Sass
  - Yeoman
  - ChicoUI
  - Mesh
